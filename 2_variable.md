## 1. type 유무

go는 type 언어이기 때문에 type을 명시적으로 선언해주어야 한다.

상수를 선언할 떄
`const name string = "hoon"`

변수를 선언할 떄
`var name string = "hoon"`

*추가
GO언어에는 변수 선언을 단축시키는 기호가 존재함.

`name : = "hoon"`

이런 식으로 써주면 name의 type을 알아서 추론한 컴파일러가 선언을 해준다. 당연히 이렇게 정해진 type을 중간에 바꿀 수 없다.
